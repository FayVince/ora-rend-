## Órarend Módosítása

Az órarend módosításához szerkeszd a `app/src/main/java/com/fayvince/orarend/model/Schedule.kt` fájlt.

### Példa - Új óra hozzáadása:

```kotlin
Lesson(
    subject = "Zene",
    teacher = "Varga A.",
    room = "Z101",
    dayOfWeek = DayOfWeek.MONDAY,
    periodNumber = 7
)
```

### Példa - Meglévő óra módosítása:

Keresd meg a megfelelő órát és módosítsd az értékeit:

```kotlin
// Régi
Lesson("Matematika", "Nagy T.", "E112", DayOfWeek.MONDAY, 1),

// Új
Lesson("Matematika", "Kiss A.", "E115", DayOfWeek.MONDAY, 1),
```

### Példa - Óra törlése:

Egyszerűen töröld a megfelelő sort a `lessons` listából.

## Megjegyzések

- Az órarend egy tipikus magyarországi középiskolai órarend alapján készült
- A nevek és termek kitaláltak, példaként szolgálnak
- Az órarend könnyen testreszabható a saját igények szerint
- A 7. és 8. óra jelenleg nincs használva, de a rendszer támogatja őket
